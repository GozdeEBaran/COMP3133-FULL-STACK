NodeJS Project 
	•	Create NodeJS project having name lab4_users_database 
	•	Install required modules to work with express and mongoose

	•	Create Users Schema in your project to declare validation 

	•	All fields are mandatory
	•	Username field must have min length >=4 and max length 100
	•	Allow only unique and valid email address 
	•	Allow only alphabets and space while entering city name
	•	Allow only valid web URL address (http or https is valid)
	•	Zip code format must be like 12345-1234 (DDDDD-DDDD, D = digit)
	•	Validate phone format like 1-123-123-1234 (D-DDD-DDD-DDD, D = digit)

	•	Create POST REST API to insert document to MongoDB database and perform validation
